# Git/GitHub体験ワークショップ ハンズオン素材

「自分のホームページを作る」想定で、Git/GitHubを体験するためのハンズオン素材です。

## フォルダ構成

```
git-hands-on/
├── workspace/        ← 作業フォルダ（ここで作業します）
├── versions/         ← 各バージョンの完成形
│   ├── v1_最初の状態/
│   ├── v2_自己紹介と連絡先/
│   ├── v3_壊れた版/
│   └── v4_派手デザイン/
└── プロンプト集/      ← Claude Codeに貼り付けるプロンプト
```

## 進め方

1. `versions/` の各バージョンの中身を、`workspace/` フォルダに**上書きコピー**します
2. Claude Codeに「コミットして」「プッシュして」「戻して」などと自然言語でお願いします
3. GitHub上で、変更履歴を視覚的に確認します

実際のコード修正はせず、「上書きコピー → Claude Codeに依頼」を繰り返すシンプルな流れで進みます。

## 各バージョンの内容

| バージョン | 内容 | 用途 |
|---|---|---|
| v1 | プロフィール写真とひとこと | 最初のコミット |
| v2 | 自己紹介・趣味・連絡先を追加 | 2回目のコミット |
| v3 | レイアウトが崩れた版 | 「戻す」体験用 |
| v4 | 派手デザインの別バージョン | ブランチ体験用 |
